# 美食查詢系統
本程式是**2025 資料庫理論**中我們小組的期末專題：美食查詢系統  

## ✨ 概述  

本系統為一個美食查詢平台，主要目的是協助使用者依據地點、價格、餐廳類型、評價等多項條件快速查詢各類型美食與餐廳資訊。  
  
系統具備會員註冊與登入功能、收藏機制，以及簡易評論與評分系統，提升使用者互動性與便利性。  
  
此外，考慮到台北捷運通勤族眾多，系統未來可延伸整合捷運路線資訊，讓使用者依照捷運站點篩選周邊美食，提高查詢效率與實用性。
